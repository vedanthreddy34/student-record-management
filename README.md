# Student Record Management System

A beginner friendly Python project that simulates a 
real world student database using just core Python.
Built as part of my internship training at 
SkillInfyTech IT Solutions.

## Project Overview
Managing student data is something every school or 
college needs. This project gave me a chance to think
like a developer and build a tool that actually solves
a real problem. The entire system runs in the terminal
and uses Python lists to store and manage records.

## Features at a Glance
- Register a new student with all their details
- Browse through all registered students at once
- Update any student information on the fly
- Remove a student record when no longer needed
- Handles empty inputs and duplicate name checks

## Technologies Used
- Python 3
- No external libraries needed

## Getting Started
Step 1 — Make sure Python 3 is installed
Step 2 — Download or clone this repository
Step 3 — Open terminal and run:
         python student_records.py
